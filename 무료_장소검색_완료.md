## 🔧 작동 방식

### 사용자 경험 (변화 없음)

```
사용자 A가 "강남역" 검색
   ↓
OpenStreetMap에서 검색 (무료)
   ↓
10개 결과 반환 (관련도순 정렬)
   ↓
사용자 A에게 결과 표시

사용자 B가 "Eiffel Tower" 검색
   ↓
OpenStreetMap에서 검색 (무료)
   ↓
결과 반환 (프랑스, 파리)
   ↓
사용자 B에게 결과 표시
```

**모든 사용자가 무료로 무제한 검색 가능!**

## 💡 기술 세부사항

### API 엔드포인트

```javascript
https://nominatim.openstreetmap.org/search?
  q=강남역&              // 검색어
  format=json&           // JSON 응답
  addressdetails=1&      // 상세 주소 포함
  limit=10&              // 상위 10개 결과
  accept-language=ko,en  // 한국어/영어 우선
```

### 응답 형식

```json
[
  {
    "place_id": "123456",
    "name": "강남역",
    "display_name": "강남역, 서울특별시, 대한민국",
    "lat": "37.4979",
    "lon": "127.0276",
    "type": "station",
    "importance": 0.8
  }
]
```

### 정렬 기준

- **importance** 값으로 정렬 (0.0 ~ 1.0)
- 높을수록 관련도가 높음
- 예: "강남역" 검색 시 강남역 (역) > 강남역 (버스정류장)

## 🚀 사용 방법

### 개발 서버 실행
```bash
npm run dev
```

### 테스트
1. 로그인 후 "Add place" 클릭
2. 검색창에 입력:
   - 한국: "강남역", "명동", "서울시청"
   - 해외: "New York", "Tokyo Tower", "Eiffel Tower"
3. 결과 확인 (10개까지 표시)
4. 장소 선택 → 친밀도/감정/기억 입력 → 만다라 생성

### 프로덕션 배포
```bash
npm run build
# dist 폴더를 호스팅 서비스에 업로드
```
